Lokbibliothek

Die Lokdaten werden aus der Datei locoData.csv gelesen. In der Tabelle unten wird gespiegelt, welche der Daten tatsächlich verwendet werden. Bspw. werden fehlerhafte Daten überlesen. Durch Vergleich der Datei (Link oben) und der Tabelle unten kann u.U. festgestellt werden, wo eine unerwartete Differenz ist.

Es können max. %MAX_LOCO_LIB_SIZE% Einträge sein (sind es mehr, werden sie ignoriert). Wenn noch freie Plätze vorhanden sind, werden Loks, die im manuellen Modus gefahren werden und nicht in der Bibliothek vorgegeben sind, hinzugefügt. Jedoch nur, solang freie Plätze vorhanden sind. Danach wird jede neue Lok an Stelle 0 gespeichert, die so zuletzt aufgerufene Lok überschreibt also eine vorher dort gespeicherte Lok.
